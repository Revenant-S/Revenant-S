## 注释

### 单行注释

```java
// 单行注释
```

### 多行注释

```java
/*
多行注释
多行注释
多行注释
*/
```

### 文档注释

```java
/**
  *@author 123
  */
```

## 标识符



1. 所有的标识符都应该以字母、$或者下划线开头

2. 不能以关键字作为变量名或方法名
3. 大小写敏感
