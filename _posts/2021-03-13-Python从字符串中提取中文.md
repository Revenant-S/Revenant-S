## 方法1
正则表达式: [\u4e00-\u9fa5], 只匹配汉字
依据汉字的Unicode码表: 从u4e00~u9fa5, 即代表了符合汉字GB18030规范的字符集
```python
import re

re.findall('[\u4e00-\u9fa5]', str_data) # 返回单个中文构成的列表
```
## 方法2
