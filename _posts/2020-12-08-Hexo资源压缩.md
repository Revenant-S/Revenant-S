### [hexo-all-minifier](https://github.com/chenzhutian/hexo-all-minifier)使用

这款工具它兼顾了hexo-neat的易用性，和gulp的功能。

<!-- more -->

#### 安装

```sh
npm install hexo-all-minifier --save
```


#### 配置

在根目录`_config.yml`里添加:

```yaml
all_minifier: true
```
